Setup Dashboard Settings MCP

MCP (AI assistant) access

Let Claude Desktop or Claude Code read and change your topics, sources, threshold and schedule through conversation.

1. Copy this into your Claude Desktop config

File location: %APPDATA%\Claude\claude_desktop_config.json. If the file already has a mcpServers section, merge this entry into it rather than replacing the file.

2. Restart Claude Desktop

Fully quit and reopen Claude Desktop (not just close the window) so it picks up the new server entry.

Available tools

This is configuration-only access -- it does not expose your favorites, notes, or chat history.
