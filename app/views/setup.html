First time setup: run the migration in supabase/001_schema.sql in your Supabase project's SQL Editor. It creates all tables and default settings.

Supabase

In your Supabase project, go to Project Settings → API. Copy the "Project URL" and the service_role secret key (not the anon/public key -- the service role key is required for the collector/filter/API to write data).

Groq (LLM scoring & agent chat)

Create a free key at console.groq.com/keys.

Reddit

Request script-app access via Reddit's API access request form to get a Client ID and Secret. Add your Reddit username -- Reddit's API rules require it for identification. Leave these blank to skip Reddit collection.

YouTube

In the Google Cloud Console, create a project (or use an existing one), enable the YouTube Data API v3, then create an API key under APIs & Services → Credentials. Leave blank to skip YouTube collection.
